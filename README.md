# 🚀 LifeTrack – Smart Event Reminder App

## 🌐 Live Demo

👉 https://life-track-blush.vercel.app/

---

## 📌 Overview

LifeTrack is a full-stack Smart Event Reminder web application that helps users manage and track important life events such as exams, travel plans, birthdays, and more.

---

## ✨ Features

* 🔐 Firebase Authentication (Login & Signup)
* ➕ Add Events
* ✏️ Edit Events
* 🗑 Delete Events
* 📅 Calendar View
* ⏳ Countdown Timer
* 🔒 Protected Routes
* 📱 Fully Responsive Design

---

## 🛠 Tech Stack

* ⚛️ React (Vite)
* 🎨 Tailwind CSS
* 🔥 Firebase Authentication
* ☁️ Firestore Database
* 🚀 Vercel (Deployment)

## ⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/your-username/lifetrack.git

2. Navigate to project:
   cd lifetrack

3. Install dependencies:
   npm install

4. Create `.env` file:
   VITE_API_KEY=your_api_key
   VITE_AUTH_DOMAIN=your_auth_domain
   VITE_PROJECT_ID=your_project_id
   VITE_STORAGE_BUCKET=your_storage_bucket
   VITE_MESSAGING_SENDER_ID=your_sender_id
   VITE_APP_ID=your_app_id

5. Run the project : 
     npm run dev
